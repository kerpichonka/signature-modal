## Description

Responsive markup layout made by Flexbox. Adaptation for laptop and mobile.
Stack of tech: BEM, Pug, SASS/SCSS.
Task manager: Gulp.

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
